Instructions

To use, write or paste a quiz in the required format in the main text field. Clicking "Download Quiz" will download a QTI .zip file suitable for import into Canvas LMS.

Only the four question types listed below are supported at this time. Note that formatting such as bold and italics will not paste in automatically; however, you can write in HTML tags if desired.

Required Formatting

The points line is optional; if omitted, the question defaults to 1 point. An asterisk before the letter of an answer choice indicates that the answer is correct.
